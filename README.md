# ActivityPool Social  
A fun and social activity-sharing mobile application built with **React Native (Expo)** and **Firebase**.  
Users can create, discover, and join local activities on a map — making it easy to meet new people and have fun.

---

## 🚀 Features

### 👤 Authentication  
- Email + password login & registration using **Firebase Authentication**  
- Profile with avatar selection  

### 🏞 Activities  
- Create activities with **title, description, date, time, and location**  
- Integrated **Google Places Autocomplete API** for searching locations  
- Join or unjoin activities  
- Activity host can delete their activity  

### 🗺 Map Integration  
- Activities displayed on map using **react-native-maps**  
- Fetches coordinates from **Google Place Details API**

### 🏠 Home Dashboard  
- Personalized welcome message with avatar  
- “Create Activity” and “Show Activities” buttons  
- Expandable list of all activities  
- Clean and modern UI

### 👤 Profile  
- View and edit user info  
- Change avatar  
- Logout
